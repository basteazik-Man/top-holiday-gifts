GiftLuxe — Static site
Structure:
- index.html (single-page preserving original categories)
- assets/css/styles.css
- assets/js/scripts.js

How to use:
1. Unzip the package.
2. Open index.html in browser.
This site keeps original categories from the uploaded index.html and implements the filtering system:
'Повод', 'Получателю', 'Цене', 'Категории'.
